**Suggested title:** Five AI Stories Worth Your Attention This Week

## 1. Anthropic Launches Claude Opus 4.7

Anthropic released Claude Opus 4.7 on April 16, 2026, bringing notable upgrades in coding performance and vision capabilities. The model now accepts images up to 2,576 pixels on the long edge (roughly 3.75 megapixels), which is more than three times the resolution of previous versions.

One detail worth paying attention to: Opus 4.7 follows instructions more literally than its predecessor, Opus 4.6. If you have existing prompts, you may need to adjust them.

The release also introduces an "xhigh" effort level, a new setting between high and maximum that gives users more control over the balance between reasoning depth and speed. Pricing stays the same at $5 per million input tokens and $25 per million output tokens. The model is available through the Claude API, Amazon Bedrock, Google Cloud's Vertex AI, and Microsoft Foundry.

**Source:** [Anthropic launches Claude Opus 4.7](https://www.streetinsider.com/Corporate+News/Anthropic+launches+Claude+Opus+4.7+with+enhanced+coding+and+vision+capabilities/26322789.html)

## 2. Runway's CEO Wants Hollywood to Bet on Volume, Not Blockbusters

Cristobal Valenzuela, CEO of AI video generation startup Runway (valued at $5.3+ billion), pitched a provocative idea: instead of spending $100 million on one feature film, studios should use AI to make 50 films with that same budget at the same visual quality.

It's not purely theoretical. The AI-assisted film "Bitcoin: Killing Satoshi" reportedly brought a $300 million estimated production down to $70 million. Major studios including Amazon and Sony Pictures are already deploying AI to reduce costs. James Cameron has publicly endorsed AI tools for maintaining blockbuster production without layoffs.

The counterargument is real, too. Critics question whether treating filmmaking as a probability game, rather than an artistic investment, actually produces anything worth watching.

**Source:** [TechCrunch: Runway CEO on AI and Hollywood](https://techcrunch.com/2026/04/16/runway-ceo-says-ai-could-help-hollywood-make-50-films-instead-of-one-100m-blockbuster/)

## 3. Canva's AI Assistant Now Calls External Tools on Your Behalf

Canva shipped a significant update to its AI design assistant. The system can now autonomously call external tools, including Slack, Gmail, Google Drive, Calendar, and Zoom, to build context and generate layered, editable designs from a single text prompt.

New features include scheduling functionality (outputs appear as drafts before posting), an improved code generator that supports HTML imports, and natural language spreadsheet creation. On the performance side, Canva says its Lucid Origin image generation model now runs 5x faster at 30x lower cost, and its 12V image-to-video model is 7x faster and 17x cheaper.

With Adobe launching its Firefly AI assistant and Figma integrating its own AI agent, the design tool space is becoming an agentic AI battleground.

**Source:** [TechCrunch: Canva's AI assistant update](https://techcrunch.com/2026/04/16/canvas-ai-assistant-can-now-call-various-tools-to-make-designs-for-you/)

## 4. DeepL Moves from Text to Voice Translation

DeepL, best known for its text translation engine, launched real-time voice translation. The initial approach works in three steps: speech to text, translation, then text back to speech. CEO Jarek Kutylowski acknowledged the core challenge is "balancing reducing latency and maintaining accurate results." The company plans to eventually build an end-to-end voice model that eliminates the text intermediary.

The product lineup includes add-ons for Zoom and Microsoft Teams (currently in early access), mobile and web conversation tools, group conversation features with QR code access for workshops, and a developer API for custom integrations like call center solutions. The system can also learn custom vocabulary, including industry-specific terms and proper names.

DeepL faces competition from Sanas ($65M funded, focused on accent modification), Camb.AI (speech synthesis and video dubbing), and Palabra (real-time translation with voice preservation).

**Source:** [TechCrunch: DeepL voice translation](https://techcrunch.com/2026/04/16/deepl-known-for-text-translation-now-wants-to-translate-your-voice/)

## 5. A Developer Woke Up to a 54,000 Euro Bill from Google's Gemini API

This one is a cautionary tale. A developer using Firebase AI Logic with an unrestricted browser API key got hit with over 54,000 euros in charges within 13 hours. The key, embedded in client-side code, was exploited for unauthorized Gemini API calls overnight.

By the time Google's cost alerts triggered, the bill was already at 28,000 euros. Google Cloud Support classified the charges as valid because the requests came from the developer's own project. The billing adjustment request was denied.

Google has since responded with several protective measures: spending caps for Tier 1 users (default $250/month), project-level spending limits, automatic detection of exposed keys, and a shift toward auth keys by default for new users.

The lesson is straightforward: never embed unrestricted API keys in client-side code, and always set spending caps before you need them.

**Source:** [Google AI Developer Forum: €54k billing spike](https://discuss.ai.google.dev/t/unexpected-54k-billing-spike-in-13-hours-firebase-browser-key-without-api-restrictions-used-for-gemini-requests/140262)
